What is WordPress Claw?

WordPress Claw is your content command center‚Äîa system that connects Google Sheets, AI, and WordPress into a seamless content creation pipeline.

Instead of juggling multiple tools, you manage everything from a spreadsheet. Add topics, track progress, and publish articles‚Äîall without leaving Google Sheets.

üí° Perfect for

Content teams, marketing agencies, bloggers, and businesses that need to produce consistent, high-quality content at scale.

The Core Idea

Traditional content workflows involve:

  • Researching topics in one tool
  • Writing in another
  • Formatting in WordPress
  • Tracking progress in spreadsheets

WordPress Claw unifies this: Your spreadsheet is your command center. Everything flows through it.

Quick Start Guide

Get your first article published in under 10 minutes:

  1. Create your account

    Sign up at wordpress-claw.com/register and complete your business profile.

  2. Connect Google Sheets

    Go to Dashboard ‚Üí Connections and link your Google account. We'll create your command center spreadsheet.

  3. Connect WordPress

    Add your WordPress site URL and credentials. We support both Application Passwords and traditional login.

  4. Add your first topic

    Open your spreadsheet, enter a topic in the first row, and set status to PENDING.

  5. Generate and publish

    Use ClawBot or the dashboard to generate your article. Watch it publish automatically to WordPress!

How the Command Center Works

Your Google Sheet is the single source of truth for your content operation. Here's how the data flows:

The Flow

  1. You add topics ‚Äî Enter article ideas in the TOPIC column
  2. Set status to PENDING ‚Äî This tells the system the row is ready
  3. ClawBot processes ‚Äî Status changes to PROCESSING during generation
  4. Article publishes ‚Äî Status becomes DONE, URL is added

Why Spreadsheets?

  • Familiar interface ‚Äî Everyone knows how to use a spreadsheet
  • Real-time collaboration ‚Äî Multiple team members can work simultaneously
  • No learning curve ‚Äî No new software to master
  • Flexible structure ‚Äî Add columns as your needs evolve

‚ö†Ô∏è Column Names

Column headers are case-insensitive but must match exactly (e.g., "Topic" and "TOPIC" both work, but "Article Topic" won't be recognized as the topic column).

Status Meanings

The STATUS column controls your content workflow. Here are the three states:

PENDING

Article is queued and ready for processing. Change to this when you want ClawBot to pick up the topic.

PROCESSING

ClawBot is actively working on this article. Don't edit the row while in this state.

DONE

Article has been generated and published. The WORDPRESS_URL column will contain the live link.

Google Sheets Setup

Connecting Google Sheets allows WordPress Claw to read your topics and write back status updates.

Setup Steps

  1. Go to Connections

    In your dashboard, navigate to Settings ‚Üí Connections.

  2. Click "Connect Google Sheets"

    You'll be redirected to Google's OAuth consent screen.

  3. Grant Permissions

    Allow WordPress Claw to access your Google Sheets. We only request the minimum permissions needed.

  4. Create or Select Spreadsheet

    Choose to create a new spreadsheet or connect an existing one.

‚ÑπÔ∏è Permissions Explained

We request read/write access to Google Sheets only. We cannot access your Gmail, Drive files, or other Google data.

WordPress Connection

Connect your WordPress site to enable automatic publishing.

Requirements

  • WordPress 5.6 or higher
  • REST API enabled (enabled by default)
  • Pretty permalinks enabled

Connection Methods

Method 1: Application Passwords (Recommended)

  1. In your WordPress admin, go to Users ‚Üí Profile
  2. Scroll to Application Passwords
  3. Enter "WordPress Claw" as the name and click Add New
  4. Copy the generated password
  5. Paste it into the WordPress Claw connection form

Method 2: Traditional Login

You can also connect using your WordPress username and password. This is less secure but works on all WordPress sites.

‚ö†Ô∏è Security Note

We recommend Application Passwords because they can be revoked independently of your main password and don't expose your actual login credentials.

GitHub for Images

WordPress Claw can generate featured images for your articles and host them on GitHub. This provides fast, reliable image delivery without bloating your WordPress media library.

Why GitHub for Images?

  • Free hosting ‚Äî GitHub serves images via CDN at no cost
  • Fast loading ‚Äî Images load quickly worldwide
  • No WP bloat ‚Äî Keeps your WordPress media library clean
  • Version control ‚Äî Track image changes over time

Setup

  1. Create a GitHub Repository

    Create a new public or private repo (e.g., my-site-images).

  2. Generate a Personal Access Token

    Go to GitHub Settings ‚Üí Developer settings ‚Üí Personal access tokens ‚Üí Tokens (classic).

  3. Add Token to WordPress Claw

    In your dashboard, go to Settings ‚Üí Connections ‚Üí GitHub and paste your token.

üí° Optional Connection

GitHub image hosting is optional. If not connected, articles will be published without featured images (or you can add them manually in WordPress).

How to Use the ClawBot Agent

ClawBot is your AI content assistant. Available on Pro plans, it can understand natural language commands and manage your content workflow.

Accessing ClawBot

Navigate to Dashboard ‚Üí ClawBot or visit /dashboard/clawbot.html.

What ClawBot Can Do

  • Generate articles from your spreadsheet topics
  • Research topics before writing
  • Answer questions about your content strategy
  • Create content calendars and schedules
  • Analyze your published content performance
üí° Getting the Most from ClawBot

Be specific in your requests. Instead of "write about marketing," try "write a 1000-word article about email marketing for SaaS companies, targeting the keyword 'SaaS email marketing best practices'."

Available Commands

Here are the commands you can use with ClawBot:

Command Description Example
generate [topic] Generate an article on the specified topic "generate an article about SEO trends"
process row [number] Process a specific row from your spreadsheet "process row 5"
process all Process all PENDING rows "process all pending articles"
status Check the status of your content pipeline "what's the status?"
research [topic] Research a topic before writing "research AI in healthcare"
schedule [topic] for [date] Schedule an article for future publication "schedule 'Holiday Gift Guide' for Dec 1"

You can also ask questions naturally. ClawBot understands context and can have a conversation about your content strategy.

Understanding the Workflow

Here's what happens behind the scenes when you process an article:

Step-by-Step Process

  1. Read Spreadsheet

    ClawBot reads your spreadsheet and identifies rows with STATUS = PENDING.

  2. Update Status

    The status changes to PROCESSING to prevent duplicate work.

  3. Research

    The AI researches the topic, gathering relevant information and understanding context.

  4. Generate Content

    Using your business profile and any specified parameters (tone, keywords), the AI writes the article.

  5. Generate Image

    If GitHub is connected, a featured image is generated and uploaded.

  6. Publish to WordPress

    The article is published with proper formatting, categories, and tags.

  7. Update Spreadsheet

    Status changes to DONE and the WordPress URL is added.

‚ÑπÔ∏è Processing Time

Most articles complete in 2-5 minutes. Complex topics or longer articles may take up to 10 minutes.
